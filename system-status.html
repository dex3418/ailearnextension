System Capabilities (Completed & Stable)

  1. Intelligent Time Tracking: Active tab time, engagement, topic-wise investment, idle detection, and detailed daily logs.
  2. Browser-Level Content Extraction: Reads educational text from active tabs, filters noise, and structures topics.
  3. AI-Based Topic Structuring: Gemini categorizes topics, summarizes content, extracts concepts, and separates theory vs. practice.
  4. Supabase Cloud Integration: Secure storage for logs, assessments, time tracking, and user-centric data tables.
  5. End-of-Day Assessments: Five compulsory questions covering both concept and applied understanding in a guided 30-minute session.
  6. Learning Analytics Engine: Computes time per topic, frequency, daily volume, performance indicators, and efficiency correlations.
  7. Smart Suggestions Engine: Recommends related topics, deeper areas, gaps to fill, and progression paths.
