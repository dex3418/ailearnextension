Developer Installation

Install the AI Learning Growth Tracker Extension

Load the production build directly from Chrome by pointing the extension loader at the compiled dist/ folder.

Installation Steps

Step 1

Open Google Chrome and navigate to chrome://extensions/.

Step 2

Toggle Developer Mode in the top-right to expose actions like Load unpacked.

Step 3

Click Load unpacked and select the dist/ folder that contains the production build (not the root project folder).

Step 4

Pin the extension by opening the Chrome extensions menu (🧩), finding AI Learning Growth Tracker, and clicking the pin icon.

Step 5

Click the extension icon and choose Open Side Panel so the tracker is always accessible while you browse.

Updating the Extension

After changing the code, reload the extension at chrome://extensions/ by clicking the Reload button on the extension card. No reinstall is necessary.

Important Notes

  • manifest.json and all assets must stay inside dist/.
  • The extension uses Manifest V3 for modern Chromium compliance.
  • No backend setup is required; API keys are entered through the UI.
  • Local storage keeps keys encrypted and secure.

Installation Complete

  • Browser-level AI Learning Tracker is active.
  • Persistent Side Panel stays visible as you study.
  • Secure in-app API configuration is available.
  • Optional Supabase sync can be configured later.
